Описание (Русский)
TGLink - это плагин для серверов Minecraft, который обеспечивает интеграцию с Telegram. Он позволяет игрокам привязывать свои игровые аккаунты к Telegram-боту, получать бонусы за первую привязку, управлять статусом блокировки аккаунта и защищать вход по IP-адресу.

Функционал
Привязка аккаунта Minecraft к Telegram через одноразовый код.

Награда за первую успешную привязку (алмазы и опыт).

Возможность блокировки и разблокировки доступа к серверу прямо из Telegram.

Система защиты: если IP-адрес игрока изменился, он автоматически перемещается в мир AUTHTG до подтверждения входа.

Получение информации об аккаунте.

Инструкция по установке
Скачайте скомпилированный jar-файл.

Поместите файл в папку plugins вашего сервера.

Перезапустите сервер.

Используйте команду /link в игре для получения кода привязки.


Description (English)
TGLink is a Minecraft server plugin that provides integration with Telegram. It allows players to link their in-game accounts to a Telegram bot, receive rewards for the first-time link, manage account blocking status, and protect their login via IP verification.

Features
Link Minecraft account to Telegram using a one-time code.

Reward for the first successful link (diamonds and experience).

Ability to block and unblock server access directly from Telegram.

Security system: if a player's IP address changes, they are automatically teleported to the AUTHTG world until the login is confirmed.

Account information retrieval.

Installation Instructions
Download the compiled jar file.

Place the file in your server plugins folder.

Restart the server.

Use the /link command in-game to obtain the linking code.



Требования / Requirements

Java 8+

Telegram Bot API

SQLite (встроенная поддержка)
